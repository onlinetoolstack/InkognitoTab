# InkognitoTab

Easily switch regular browsing between and incognito modes with the click of a button using InkognitoTab extension

Open the existing tab within an incognito window effortlessly by simply clicking the "InkognitoTab" button.

InkognitoTab is a simple extension that opens your current tab in an incognito (private browsing) or window. No unnecessary permissions or tracking!

Keep the original tab while opening the incognito one.  
Open the incognito tab in a maximized view.

Effortlessly switch between browsing modes by incorporating these shortcuts into your routine, providing a fast and user-friendly way to navigate your tabs with InkognitoTab.

Utilize the following keyboard shortcuts for a seamless experience:

Windows: Ctrl+Q  
Mac: Command+Q  
Chrome OS: Ctrl+Q  
Linux: Ctrl+Q
